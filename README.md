# 408-Master

> 408考研学习指南 - 一站式计算机考研复习工具

## 📖 简介

408-Master 是一款专为计算机考研（408统考）设计的桌面学习应用，集成了刷题、视频学习、笔记、学习管理等功能，帮助考研党高效备考。

## ✨ 核心功能

### 📚 四大科目学习
- **数据结构** - 线性表、树、图、排序、查找等
- **计算机组成原理** - CPU、存储系统、指令系统、总线等
- **操作系统** - 进程管理、内存管理、文件系统、I/O等
- **计算机网络** - 网络体系结构、TCP/IP、应用层协议等

### 🎯 刷题系统
- 王道选择题题库（含解析）
- 综合题/大题练习
- 按章节分类刷题
- 错题记录与复习

### 🎬 视频学习
- 集成视频播放器
- 支持本地视频管理
- 学习进度追踪

### 🛠️ 学习工具
- **番茄钟** - 专注学习计时，支持悬浮窗
- **学习热力图** - 可视化学习打卡记录
- **倒计时日历** - 考研倒计时
- **Markdown笔记** - 支持公式、代码高亮
- **思维导图** - DrawIO集成
- **语音转文字** - 音频笔记转录

### 📊 学习管理
- 学习计划制定
- 学习进度统计
- 每日打卡签到
- 成就系统

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装运行

```bash
# 克隆项目
git clone https://github.com/GalaxyPoke/408-Master.git
cd 408-Master

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或单独启动前端
npm start
```

### 打包桌面应用

```bash
# 构建 Electron 应用
npm run electron-build
```

## 📁 项目结构

```
408-Master/
├── src/                    # React 前端源码
│   ├── components/         # 页面组件
│   │   ├── tools/          # 学习工具组件
│   │   ├── notes/          # 笔记相关组件
│   │   └── video/          # 视频播放组件
│   └── data/               # 题库数据
├── server/                 # 后端服务
├── public/                 # 静态资源
│   ├── videos/             # 学习视频
│   └── audios/             # 音频资源
├── doc/                    # 王道做题本文档
└── assets/                 # 应用图标
```

## 🔧 技术栈

- **前端**: React 18 + Tailwind CSS
- **桌面端**: Electron
- **后端**: Express + SQLite
- **其他**: Markdown编辑器、DrawIO、Tesseract OCR
